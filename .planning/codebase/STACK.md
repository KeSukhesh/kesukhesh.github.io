# Stack

| Layer | Technology | Version |
|-------|-----------|---------|
| Framework | Astro | 5.x |
| Styling | Tailwind CSS | 4.x |
| Content | Astro Content Collections | Built-in |
| Syntax Highlighting | Shiki | Built into Astro |
| Search | Fuse.js | Latest |
| Fonts | Libre Baskerville, Space Mono | Google Fonts |
| Deployment | GitHub Pages | Via Actions |
| Package Manager | npm | Latest |

## Design Tokens
- **Colors**: Black (#0a0a0a), Off-white (#f5f5f0), Forest green (#2d5a3d), Gray scale
- **Typography**: Libre Baskerville (headings + body), Space Mono (code)
- **Animations**: CSS transitions + IntersectionObserver, scroll-snap
